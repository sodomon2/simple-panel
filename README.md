# simple-panel
A simple panel 

## Build and install

```
git clone https://github.com/sodomon2/simple-panel.git
meson setup build
ninja -C build
[sudo] ninja -C build install
```

## Dependencies

- [GTK4](https://www.gtk.org/)
- [Gio](https://developer.gnome.org/gio/stable/)
- [GtkLayerShell](https://github.com/wmww/gtk4-layer-shell)
